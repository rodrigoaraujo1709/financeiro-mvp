# 💰 Financeiro MVP

Sistema financeiro pessoal desenvolvido para centralizar:
- controle financeiro
- contas fixas e variáveis
- dívidas e recebimentos
- anexos/comprovantes
- organização financeira pessoal

---

# 🚀 Objetivo do Projeto

Este projeto nasceu com os objetivos de:

- praticar desenvolvimento moderno
- reaprender arquitetura de software
- estudar tecnologias atuais
- explorar IA aplicada ao desenvolvimento
- construir um MVP real com potencial de evolução

---

# 🧠 Visão do MVP

O sistema permitirá inicialmente:

## ✅ Controle financeiro
- receitas
- despesas
- saldo mensal
- contas fixas
- contas variáveis

## ✅ Gestão de pagamentos
- contas pagas
- contas pendentes
- vencimentos

## ✅ Controle de dívidas
- valores a pagar
- valores a receber
- pessoa/empresa relacionada
- status da dívida

## ✅ Gestão documental
- upload de comprovantes
- organização de anexos
- vínculo com lançamentos financeiros

## Frontend

- PostgreSQL
